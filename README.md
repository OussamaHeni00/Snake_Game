# 🐍 Snake_Spiel mit C Programming Language (Mingw64 GCC) / SDL2 ( Simple Direct Media Layer) / Visual Studio Code 

## 🎯 Demo

https://github.com/user-attachments/assets/cefe17da-68c7-4060-a044-c53e75600478

## 🎯 Funktionen

- Klassisches Snake-Spiel mit Bewegung per Pfeilasten
- Punkte sammeln durch das Fressen von Nahrung
- Game-Over-Zustand bei Kollision mit sich selbst oder der Wand
- Anzeige von Punktzahl und Spielzeit
- Hintergrundmusik und Soundeffekte (Fressen/Kollision)
- Moglichkeit, das Spiel neu zu starten (Taste **N**) oder zu beenden (Taste **Q**)

  ## 🎯 Steuerung

  |               |  Funktion                     |
  |...............|...............................|
  | ↓ ↑ → ←       |  Bewegung der Schlange        |
  | N             |  Neues Spiel starten          |
  | Q             |  Spiel beenden                |

  ## 🎯 Abhangigkeiten

- SDL2
- SDL2_ttf
- SDL2_mixer

  ## 🎯 Kompilierung

  make
  ./main.c
